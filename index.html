网站介绍

这是一个在线的编辑或查看，适用于LuckPerms插件



这个站点托管了许多额外的web应用程序,与插件一起工作.


这些应用程序是为所有用户设计的，即使是那些没有能力在自己的web服务器上安装/托管应用程序的用户.


  • 权限编辑器

    可以修改某个用户或者某个权限组的权限,并可以保存在服务器内

  • Verbose Viewer

    A web based viewer for verbose recording logs.

  • Tree Viewer

    A web based viewer to visualise tree permission structures.
