Web Applications

Online editors/viewers built into the plugin.



This site hosts a number of extra web applications which work with the plugin.


These applications are designed to work for all users, even those without the ability to install/host an application on their own web server.


  • Web Editor

    A web based permissions editor for data stored in the plugin.

  • Verbose Viewer

    A web based viewer for verbose recording logs.

  • Tree Viewer

    A web based viewer to visualise tree permission structures.
