Web applicaties

Online editors / kijkers ingebouwd in de plugin.



Deze site bevat een aantal extra webtoepassingen die met de plugin werken.


Deze applicaties zijn ontworpen om te werken voor alle gebruikers, zelfs diegenen die geen applicatie op hun eigen webserver kunnen installeren / hosten.


  • Web Editor

    Een op het web gebaseerde machtigingseditor voor gegevens die zijn opgeslagen in de plugin.

  • Verbose Viewer

    Een webgebaseerde viewer voor uitgebreide registratie-logboeken.

  • Tree Viewer

    Een webgebaseerde viewer om structuurstructuren voor tree structures visualiseren.
